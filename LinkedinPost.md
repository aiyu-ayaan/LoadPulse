# LinkedIn Post Draft

Built **LoadPulse**: a project-based load testing dashboard for websites and apps.

It lets teams:
- create multiple projects with their own URLs
- run k6-powered tests in parallel
- watch live performance data on a simple dashboard
- review test history and reports
- share project access with teammates by email
- support sign-in with local auth and optional GitHub login

One part I cared about a lot was making the experience understandable even for non-technical users, not just engineers reading raw performance numbers.

Tech stack:
- React + Vite
- Node.js + Express
- MongoDB
- Socket.IO
- k6

It is still **in development**, so there may still be bugs, rough edges, or incomplete flows while I keep improving it.
